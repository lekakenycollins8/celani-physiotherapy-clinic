# Design Inspirations & Creative Direction

# Purpose

This document guides the visual and UX direction of the CELANI Physiotherapy & Sports Injuries Clinic website.

The goal is NOT to copy existing websites.

Instead, these references should guide:

- Visual hierarchy
- UX patterns
- Typography direction
- Premium healthcare positioning
- Layout composition
- Motion behavior
- Spacing systems
- Conversion-focused design decisions
---

# Overall Creative Direction

The website should feel like:

“A premium modern rehabilitation clinic focused on professional recovery care.”

The experience should combine:

- Premium healthcare aesthetics
- Human-centered warmth
- Editorial sophistication
- Calm professionalism
- Modern rehabilitation branding
- Conversion-focused UX
Avoid:

- Generic AI-generated designs
- Corporate hospital aesthetics
- Overly tech-startup interfaces
- Wellness spa aesthetics
- Fitness gym branding
- Template-heavy layouts

---

# Motion Design Direction
Motion should feel:

- Calm
- Elegant
- Soft
- Intentional

---

## Allowed Motion

- Fade-up reveals
- Soft stagger animations
- Gentle hover transitions
- Smooth mobile menu transitions
---

## Avoid

- Excessive animation
- Fast movement
- Bouncy effects
- Aggressive parallax
- Overly playful motion

Healthcare websites should prioritize trust and calmness over entertainment.

---

# Design Philosophy Summary

The final website should feel like:

- Premium healthcare
- Modern rehabilitation expertise
- Calm professionalism
- Human-centered recovery care
