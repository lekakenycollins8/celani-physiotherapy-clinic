# Service Page Structure

Each service page should follow the same reusable structure.

---

# 1. Breadcrumb Banner

Include:

Home / Services / Service Name

Use:

- Minimal premium styling
- Soft gradient background
- Elegant spacing

Avoid large hero sections.

---

# 2. Intro Section

Include:

- Service overview
- Recovery goals
- Conditions addressed

---

# 3. Conditions Treated

Use bullet lists or structured cards.

Example:

- Muscle injuries
- Joint pain
- Sports injuries
- Mobility limitations

---

# 4. Treatment Approach

Explain:

- Assessment methods
- Rehabilitation techniques
- Exercise therapy
- Personalized care

---

# 5. Recovery Process

Use visual steps or timeline.

---

# 6. Benefits Section

Focus on outcomes:

- Reduced pain
- Improved mobility
- Faster recovery
- Increased strength
- Injury prevention

---

# 7. Related Services

Use internal linking cards.

Purpose:

- Improve crawlability
- Increase session duration
- Support SEO silos

---

# 8. CTA Section

Use:

- Book Appointment button
- WhatsApp CTA

Avoid:

- Large contact forms
- FAQ duplication
- Testimonial duplication

These exist globally on homepage.

---

# SEO Strategy

Each service page should target:

- Service keyword
- Nairobi geo modifier
- Rehabilitation-related long-tail keywords

---

# Content Direction

Tone should remain:

- Professional
- Calm
- Reassuring
- Informative
- Non-technical for patients
